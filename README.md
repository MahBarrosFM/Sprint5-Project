# Vehicle Sales Dashboard

## Descrição

Este projeto consiste em um aplicativo web desenvolvido com **Streamlit** para realizar uma análise exploratória de um conjunto de dados de anúncios de venda de veículos.

O aplicativo permite visualizar informações do conjunto de dados por meio de gráficos interativos.

## Funcionalidades

* Exibição de um histograma da quilometragem dos veículos (`odometer`);
* Exibição de um gráfico de dispersão entre quilometragem (`odometer`) e preço (`price`);
* Interface interativa desenvolvida com Streamlit.

## Tecnologias utilizadas

* Python
* Pandas
* Plotly Express
* Streamlit
